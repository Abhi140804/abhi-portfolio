HOSTEL MANAGEMENT SYSTEM

The Hostel Management System is a Java-based application using Swing for the graphical interface and MySQL as the database to store and manage hostel data. It simplifies hostel administration by providing a user-friendly interface for tasks like student registration, room allocation, and viewing student information.

VACCINATION SYSTEM

The Vaccination System is a web-based application with a C backend, using HTML, CSS, and JavaScript for the frontend to provide an intuitive and accessible interface for users and healthcare administrators. This system is designed to streamline the entire vaccination process, from user registration and appointment scheduling to vaccine inventory tracking and record maintenance. By combining a structured backend with a user-friendly frontend, it ensures both data integrity and ease of use.
